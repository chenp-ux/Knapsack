# D{0-1}KP背包问题求解器

## 项目简介
本项目基于“动态规划算法”实现D{0-1}KP（折扣0-1背包问题）的最优解求解，包含数据读取、可视化绘图、比值排序、结果保存等核心功能，并提供友好的图形用户界面（GUI）。

### 实验背景
本项目为软件工程课程个人实验项目，目标是：
1. 掌握软件项目个人开发流程（PSP）；
2. 掌握Github发布软件项目的操作方法；
3. 实现经典组合优化问题的算法求解与工程化落地。

## 功能特性
### 基础功能
✅ 读取自定义格式的D{0-1}KP数据集（txt格式）  
✅ 绘制物品重量-价值分布散点图  
✅ 按项集第三项的价值/重量比非递增排序  
✅ 动态规划算法求解最优解（最大价值）  
✅ 结果保存为TXT/Excel格式  
✅ 基于tkinter的可视化操作界面  

### 扩展功能
🔧 算法性能计时（统计求解耗时）  
🔧 异常处理（数据格式校验、文件不存在等）  
🔧 多平台兼容（Windows/Linux/macOS）

数据格式说明
d01kp_data.txt数据集格式：
第一行：背包最大载重量（整数）；
后续每行：一个项集的 3 个物品数据，格式为w1,v1,w2,v2,w3,v3（英文逗号分隔）；
要求：每个项集的第三个物品满足v3=v1+v2，w3<w1+w2（D {0-1} KP 定义）。
示例：
20
2,3,3,4,4,7
5,6,4,5,8,11
7,9,6,8,12,17
编程规范
遵循《阿里巴巴 Python 开发手册》（适配实验要求的 Java/C++ 规范），核心规则：
缩进：4 个空格，禁止 tab；
命名：变量 / 函数用小写 + 下划线，类用大驼峰；
行宽：每行不超过 120 字符；
注释：类 / 函数必须有文档字符串，关键逻辑加单行注释；
空行：函数 / 类之间空 2 行，逻辑块之间空 1 行；
操作符：加减乘除等操作符前后各空 1 个空格。
